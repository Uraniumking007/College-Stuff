Practical 10 - Basic File Operations using Node.js

Demonstrating fundamental file system operations using Node.js built-in fs module

Introduction to File System Operations

Node.js provides a built-in fs module for working with the file system. It allows you to create, read, update, and delete files and directories.

To run Node.js file operation programs:
  1. Install Node.js from nodejs.org
  2. Save code in a .cjs file (e.g., 01-create-files.cjs)
  3. Open terminal and run: node filename.cjs

📋 How to Run File Operation Examples
📁 Location: backend/practical10/

Step-by-Step Instructions:

  1. Open terminal/command prompt
  2. Navigate to the practical10 directory: 
    cd backend/practical10
  3. Run any example: 
    node 01-create-files.cjs
  4. Watch the terminal for output and file creation messages
  5. Check the data/ folder to see created/modified files
🚀 Quick Start Commands:

Run:

cd backend/practical10
node 01-create-files.cjs

Output: Creates a data/ folder with example files (text, JSON, binary)

Run:

cd backend/practical10
node 02-read-files.cjs

Note: Run 01-create-files.cjs first to generate test data

Run:

cd backend/practical10
node 03-append-files.cjs

Output: Creates data/app.log with various log entries

Run:

cd backend/practical10
node 04-file-info.cjs

Output: Displays file statistics, sizes, and modification times

Run:

cd backend/practical10
node 05-delete-files.cjs

Warning: This will delete and rename files. Run 01-create-files.cjs first to generate test files

Run:

cd backend/practical10
node 06-directories.cjs

Output: Creates and manages various test directories

Run:

cd backend/practical10
node 07-watch-files.cjs

Note: This example runs for ~15 seconds demonstrating various file changes automatically
