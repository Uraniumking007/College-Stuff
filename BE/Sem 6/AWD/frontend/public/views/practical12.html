About Docker Integration

🐳 Docker Compose Setup

This application uses Docker Compose to orchestrate three containers:

  • MongoDB: Database container on port 27017
  • Backend API: Express.js server on port 3000
  • Frontend: AngularJS application on port 4200

🔧 Architecture

  • Frontend communicates with Backend via REST API
  • Backend connects to MongoDB using Mongoose
  • All services are containerized and networked
  • Data persists in Docker volume for MongoDB

🚀 Running with Docker

# Build and start all services
docker-compose up --build

# Stop all services
docker-compose down

# View logs
docker-compose logs -f
